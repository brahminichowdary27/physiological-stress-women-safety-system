Physiological Stress Based Automatic Women Safety Alert System
Problem

In many emergency situations, victims are unable to manually call or send alerts. Existing safety apps depend on user interaction, which may fail during panic, unconsciousness, or physical restraint.

Proposed Solution

A wearable embedded system that automatically detects abnormal physiological stress conditions and sends an emergency alert with live location to predefined contacts.

System Overview

The device continuously monitors physiological signals.
When values cross a danger threshold, the controller triggers an automatic alert without requiring user action.

Flow:
Sensor Data ‚Üí Microcontroller Processing ‚Üí Threshold Detection ‚Üí GSM Alert + Location Sharing
